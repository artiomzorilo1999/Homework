Главная Галерея

Команда которые мы ростатривали

  1. git status - что изменилось
  2. git add . - добавляем все файлы которые изменили
  3. git commit -m "Название комита(заметка для нас что изменили)" - сохраняем выбраный текст(Не открывая редактор)
  4. git push - отправка на сайт (Приусловии если у нас толькло одна ветка)
  5. git push origin main - отправка в избраную ветку (main - это и есть ветка, вместо main можна выберать другие ветки если у нас они есть)
  6. git remote -v - В каком репозитории мы сохраняем
  7. git branch - просмотр веток
  8. git checkout "Название ветки" - переход на новую ветку (Если мы хотим перейди на новую ветку, мы должны убедится что сохранили все на поточной )
  9. git branch -b Название ветки которую сделали
  10. git push origin HEAD - Где находится голова проэкта
